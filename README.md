# Break Clock

一個可愛的桌面久坐提醒小工具，偵測你持續使用電腦超過45分鐘就提醒你站起來伸展。

## 功能
- 偵測滑鼠/鍵盤活動，持續使用45分鐘自動提醒
- 離開5分鐘自動重置計時
- 像素風粉色介面
- 支援 Mac / Windows

## 安裝需求

Mac:
pip3 install pillow pyobjc-framework-Quartz

Windows:
pip install pillow pywin32

## 使用方式
1. 把 app.py 和圖片檔案放在同一個資料夾
2. 執行 python3 app.py
3. 視窗會出現在右下角，開始計時
